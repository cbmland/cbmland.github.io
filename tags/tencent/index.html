用QQ来安装虚拟打印机，方便写作设计。

Author: cbm Edit

21 二 2005

就要毕业了，都在忙简历了，常有朋友问我一些关于打印的问题，如分辨率，分页显示等。其实我也不是很了解，因为我没有打印机，不过可以用虚拟软件来实现打印设置的。就没有说其他的专业虚拟软件，来个超级简单的方法，:P 我想腾讯QQ大家应该都在用吧，到im.qq.com下载最新的QQ安装程序，安装一次QQ，在你的控制面板里面的打印机和传真就有了一台Tencent Virtual Printer虚拟打印机了。一些相关打印前的设置预览就可以实现了。方便自己设计了。
(以上方法在XPsp2+QQ2004II上通过)
重启之后虚拟打印机会被删除，不过还是有疑问，真不知道腾讯为什么要安装这个虚拟打印机？ )
