千兆网卡直连 – 网络不再是瓶颈

Author: cbm Edit

12 三 2009

前面有写换了新的工作站《欢送爱机 迎来新机Dell 740MT》，其实一直让我推迟换新机的一个原因是笔记本需要通过1394火线连接工作电脑共享上网，以及之间的高速的文件共享传输。
说起来也有原因，公司网络有严格网络认证，需要安装一堆废品软件，一旦安装了会严重影响电脑速度，于是有了《禁止 Symantec Protection Agent 运行的一些方法》和《用1394火线上网》等。
然而在XP系统下，系统可以将1394接口当为以太网设备，可以通过一根1394线就可以将2台电脑组网连接起来，带宽速率可达400Mbps，而新出的操作系统Vista和Windows 7再也不被支持了，传说苹果也降低了1394火线的支持度。

成功升级 Windows Mobile 6.0

Author: cbm Edit

9 九 2007

一直在用多普达577w，对于wm5.0系统的界面我个人认为是比较丑陋的。微软的wm6发布了有半年的样子了。但577w一直没有相应的wm6能刷。在放弃中，几乎都忘记这件事情。昨天逛了一下ioicn和酷讯，才发现577w可用的wm6手机rom已经遍地都是了。一个个的比较中，选择了很有名气EA大哥的”Tornado WM6 宋体、无输入法、完整无删减最终版本 20070903“。由于之前就是使用的EA的rom，什么解锁，SID什么的统统不需要了。下载下来直连上USB开刷。十分钟内就上新系统了。哈哈。
我的577w终于用上了梦寐以求的wm6.0系统。感觉不错哦。不光界面比wm5漂亮多了，而且加入了很多人性化的支持。看来交互设计职业现在越来越有所作为了。

久等的iPod Video终于来了

Author: cbm Edit

22 十 2006

上周四终于收到了招行寄出的iPod Video 30G
这个叫爽啊.30G的空间,足够把电脑上的音乐都放上去了.而且复制歌曲的速度巨快,5G空间的music十分钟就可以装满.
不过现在用上几天的感受的觉得iPod这东西很娇贵,如果从性价比来说不是很高了,就和我一年前的mp3播放器相比,除了屏幕大,容量大,复制音乐快,其他地方都要差.
插着usb不能播放音乐了,想要边充电边听音乐都不行.
