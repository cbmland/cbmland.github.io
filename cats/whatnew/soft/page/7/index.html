Dreamweaver CS3 集成Ajax框架Spry

Author: cbm

4 八 2007

Spry 框架是新版的DW CS3的新增功能。Spry 框架是一个 JavaScript 库，Web 设计人员使用它可以构建能够向站点访问者提供更丰富体验的 Web 页。有了 Spry，就可以使用 HTML、CSS 和极少量的 JavaScript 将 XML 数据合并到 HTML 文档中，创建构件（如折叠构件和菜单栏），向各种页面元素中添加不同种类的效果。在设计上，Spry 框架的标记非常简单且便于那些具有 HTML、CSS 和 JavaScript 基础知识的用户使用。
你只指定一个XML数据集。就能通过Spry来指定循环对象。包括DIV、SPAN标签。
使用XML数据源时也非常直观。可以参看XML数据结构。
