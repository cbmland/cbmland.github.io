暴强的Ubuntu 8.04 无需改动硬盘引导和现有分区 安装在Windows环境

Windows XP SP3 简体中文版 For Acer Aspire 4720G »
Author: cbm

25 四 2008

Ubuntu 8.04 昨天4-24如期而至，我尝试了他的最新在Windows下的安装模式，发现非常的暴强。

Ubuntu 8.04 无需改动现有Windows的引导和现有的分区，就可以安装到你的机器上面，而且安装速度比光盘安装更快，做一个较好的理解：就像vmware一样，可以使用硬盘文件来安装，而这个硬盘文件可以放在你Windows下载任何盘下，只要有足够的空间，默认linux硬盘文件是15G空间分配。根本就不用什么物理上的分区，linux初学者可是可以放下分区的顾虑，安心的使用linux和windows共存了，这会可是相当的降低了linux的安装门槛。这是里程碑，是革命啊！

而引导系统会在c:\boot.ini文件添加一个Ubuntu的启动项，不会更改现有的Windows模式，选择Ubuntu后引导才会传递到正统的linux启动加载GRUB上，开始了上述的，15G的，虚拟Linux硬盘的Ubuntu启动。

Ubuntu 太伟大了~
