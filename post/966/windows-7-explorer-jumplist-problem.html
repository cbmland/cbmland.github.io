Windows 7 资源管理器Jumplist失效的解决办法

这个问题在工作机上出现了2次了，为了下次快速排除，把解决方案存档一下。

转到下面这个路径

Shell
1
%APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations

找到下面名字的文件。

Shell
1
1b4dd67f29cb1962.automaticDestinations-ms

出现问题的时候文件大小一般在4~6M，可以考虑改名备份或者直接删除，注销之后即可恢复正常。
