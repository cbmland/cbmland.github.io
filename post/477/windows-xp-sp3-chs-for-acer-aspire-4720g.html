操作步骤：（新手指引）

  1. 下载本光盘RAR压缩文档。
  2. 解压出ISO文件（如果从RAR压缩包中解压出来没有错误提示即可认为ISO文件是完整正确的。）
  3. 用Nero或其他刻录软件打开ISO文件开始刻录映像到空白CD光盘，直到刻录完毕。（刻录时勾选“效验写入数据”，以保证刻录盘的正确性）
  4. 重启，当出现Acer屏幕时按下F12键，在启动列表中选择“IDE CD”设备。
  5. 出现“Press any key boot from CD ….”按下任意键进入安装程序。
  6. 恭喜，全新的Wndows SP3即刻到来。
