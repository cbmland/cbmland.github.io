小技巧:在Flex builder 3 中创作纯AS的AIR项目

众所周知，Flex builder 3 beta3版本中已经集成了AIR的SDK，可以开发基于Flex框架的AIR应用程序了。但是使用了Flex框架编译出的SWF有点大。个人很喜欢Action Script Project（俗称纯AS项目）编译出的SWF。文件十分小巧。但如果你想在Flex builder 3 beta3中创建纯AS项目来做AIR的话会可能遇到一些困难。

当然在Flash CS3中也可以制作Action Script Project类似的。但是代码提示功能和调试功能贫弱。还是很想在Flex builder开发。发了一番时间折腾后，找到了如何在Flex Builder 3中开发和调试纯粹AS的AIR项目的小技巧。今天抽时间来整理一下。

如果你对这个感兴趣，请往下看。

1）新建Flex Preoject，选择AIR类型。直接finish

2）新建Action Script Class，同样命名为“pureASProject.as”

3）删除自动创建的pureASProject.mxml

4）把pureASProject.as项目的默认运行文件

5）编辑pureASProject-app.xml文件（重点）

<?xml version="1.0" encoding="UTF-8"?>
<application xmlns="http://ns.adobe.com/air/application/1.0.M6">
 
    <id>pureASProject</id>
 
    <filename>pureASProject</filename>
 
    <name>pureASProject</name>
 
    <version>v1</version>
 
    <description>pureASProject</description>
 
    <copyright>cbmland.com</copyright>
 
    <initialWindow>
 
        <content>[This value will be overwritten by Flex Builder in the output app.xml]</content>
        
        <title>pureASProject</title>
 
        <systemChrome>none</systemChrome>
 
        <transparent>true</transparent>
 
        <visible>true</visible>
 
    </initialWindow>
 
    <customUpdateUI>false</customUpdateUI>
    
    <allowBrowserInvocation>false</allowBrowserInvocation>
 
</application>

6）使用AIR中的API，尽情创作。现在就可以在Flex Builer中运行和调试纯AS项目了。

7）打包发布
